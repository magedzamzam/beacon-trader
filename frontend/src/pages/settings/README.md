# `frontend/src/pages/settings`

Full-page Settings views that are substantial enough to live on their own (as
opposed to the placeholder catalog in `components/settings/placeholders.jsx`).

| File | View | Purpose |
|------|------|---------|
| `Currency.jsx` | `currency` | Currency & FX settings — per-symbol/account conversion overrides used by sizing. |

Registered in `lib/nav.jsx` under Settings → Trading, like any other leaf.
